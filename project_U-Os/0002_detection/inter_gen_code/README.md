## Tasks in this folder

1. initialize (make directories):  
   `./init.sh`

2. generate data for the 100 data sets:  
   `python3 inter_gen.py 100`  

3. pack up the data:  
   `./pack.sh`  

4. all-in-one script:  
   `./script_all.sh`  
   
5. clean the generated data to restart:  
   `./clean.sh`
